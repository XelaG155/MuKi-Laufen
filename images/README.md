## Bildanforderungen:
- Format: JPG oder PNG
- Qualität: Hoch (für Web optimiert)
- Stil: Freundlich, hell, einladend
- Personen: Mütter/Väter mit Kindern im Alter von 3-6 Jahren
- Umgebung: Sporthalle oder Outdoor-Bereich

## Bildquellen-Empfehlungen:
1. **Eigene Fotos**: Aufnahmen von tatsächlichen MuKi-Kursen (mit Einverständnis)
2. **Stock-Fotos**: 
   - Unsplash.com (kostenlos)
   - Pexels.com (kostenlos)
   - Shutterstock.com (kostenpflichtig)
3. **Suchbegriffe für Stock-Fotos**:
   - "mother child exercise"
   - "family fitness"
   - "kids gymnastics"
   - "parent child sports"
   - "toddler physical activity"

## Temporäre Platzhalter:
